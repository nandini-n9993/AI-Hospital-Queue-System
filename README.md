# 🚑 AI-Hospital-Queue-System
AI-based multi speciality hospital queue management system using flask

##  Problem Statement
In many hospitals, patient queues are handled manually or on a first-come-first-serve basis.  
This often leads to:
-  Emergency patients waiting too long
-  Inefficient time management
-  Overloaded doctors handling mixed cases
-  Poor patient experience

There is a need for a **smart system** that prioritizes patients based on urgency and condition.

---

##  Solution
This project introduces an **AI-based Hospital Queue Management System** that:
- Prioritizes patients using a scoring algorithm
- Detects emergency cases automatically
- Assigns patients to appropriate specialists
- Allows doctors to manage queues efficiently

---

##  How It Works
1. Patient enters details (age, symptoms, severity, chronic condition)
2. System calculates **priority score**
3. Emergency patients are automatically boosted 🚨
4. Patients are sorted based on priority
5. Assigned to respective **specialist queue**
6. Doctor marks patient as **Done after consultation**
